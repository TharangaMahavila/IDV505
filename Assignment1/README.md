## How to run the program

1. Clone the repository
    - git clone https://github.com/TharangaMahavila/IDV505.git<br>
    - cd Assignment1
2. Install dependencies
    - You should have python running in your environment<br>
    - if not see the official installing page of python (https://www.python.org/downloads/)
3. Usage (You need to run as modules)
    - python3 -m Lecture2.nsum_test
    - python3 -m Lecture3.sort_algorithm_test
